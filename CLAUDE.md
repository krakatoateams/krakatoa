# Krakatoa Landing Page

## Project Overview
Krakatoa is an AI-powered platform for content creators, providing tools for video generation, product photography, and social media automation.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript

## Build & Development
- **Install dependencies**: `npm install`
- **Run development server**: `npm run dev`
- **Build for production**: `npm run build`
- **Linting**: `npm run lint`

## Project Structure
- `app/`: Next.js App Router directory (Pages, Layouts, CSS)
- `lib/`: Shared utility functions and database logic
- `public/`: Static assets (images, icons)

## Guidelines
- Follow Next.js 14 App Router patterns.
- Use Tailwind CSS for all styling.
- **Design Philosophy**: Premium, dark-mode first, glassmorphism, and high-quality typography.
- **Animations**: Use custom Tailwind animations (like `fade-in`) and transitions for a smooth UX.
- **Icons**: Use Lucide icons exclusively for consistency.
- Ensure all components are responsive and accessible.
- Maintain clean, self-documenting code.
