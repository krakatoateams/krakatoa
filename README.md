# Krakatoa 🌋

Krakatoa is a multi-tool AI platform designed for content creators. It provides a suite of AI-powered tools to automate social media, generate high-quality content, and streamline workflows—all in one place.

## 🚀 Tools Suite

Current and upcoming tools include:

- **ReelsGen**: Generate faceless video reels with AI narration and captions.
- **Product Photo**: Transform any product image into a professional studio shot.
- **Scheduler**: Schedule and automate your social media posts across platforms.
- **IG Automation**: Auto-generate and post Instagram content on a schedule.

## 📁 Project Structure

This repository uses a monorepo structure:

- `apps/landing`: The main Next.js landing page.
- `apps/ReelsGen`: AI-powered video generator tool.

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript

## 🚦 Getting Started

To run the landing page locally:

```bash
cd apps/landing
npm install
npm run dev
```

To explore ReelsGen:

```bash
cd apps/ReelsGen
npm install
npm run dev
```
